# LazyGeo

QGIS плагин для подготовки участков недр из лицензий Роснедра.

Берёт Excel с координатами → создаёт проект QGIS с подложкой, точками и полигоном → сохраняет SHP и DXF.

## Быстрый старт

1. Скачай `lazygeo.zip` из Releases
2. QGIS → Плагины → Управление и установка плагинов → Установить из ZIP
3. Открой плагин через Плагины → LazyGeo
4. Выбери Excel → Запустить

Если нет pandas/openpyxl — плагин покажет инструкцию по установке.

## Формат Excel

7 колонок, координаты в ГСК-2011. Заголовки не нужны — сразу данные с первой строки:

```
1  | 54 | 30 | 47,102 | 59 | 54 | 28,465
2  | 54 | 30 | 15,220 | 59 | 56 | 31,257
```

Порядок колонок: `№ | Широта ° | ' | " | Долгота ° | ' | "`

## Требования

- QGIS 3.16+
- pandas, openpyxl (pip install pandas openpyxl)
