# LazyGeo

QGIS плагин для подготовки участков недр из лицензий Роснедра.

Берёт Excel с координатами угловых точек → создаёт в QGIS слой точек и полигон участка. Дальше это можно экспортировать во что нужно стандартными средствами QGIS (SHP, DXF, GeoJSON и т.д.).

## Быстрый старт

1. Скачай `LazyGeo-x.x.zip` из Releases
2. QGIS → Плагины → Управление и установка плагинов → Установить из ZIP
3. Открой плагин через Плагины → LazyGeo
4. Выбери Excel → при необходимости поправь название слоя → Запустить

## Формат Excel

7 колонок, координаты в ГСК-2011 в формате ГМС (градусы-минуты-секунды). Заголовки не нужны — данные сразу с первой строки:

```
1  | 54 | 30 | 47,102 | 59 | 54 | 28,465
2  | 54 | 30 | 15,220 | 59 | 56 | 31,257
```

Порядок колонок: `№ | Широта ° | ′ | ″ | Долгота ° | ′ | ″`

Зона Гаусса-Крюгера определяется автоматически по средней долготе точек.

## Требования

- QGIS 3.16+

pandas и pyproj уже входят в стандартную поставку QGIS. openpyxl поставляется вместе с плагином — ничего доустанавливать не нужно.
